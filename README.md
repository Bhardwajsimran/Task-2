# To-Do List Web App

A simple To-Do List web application using Vanilla JavaScript, HTML, and CSS.

## Features
- Add new tasks
- Mark tasks as complete
- Remove tasks
- Responsive and clean UI

## How to Run
1. Open `index.html` in your browser (use Live Server for best experience).

## Tech Stack
- HTML
- CSS
- JavaScript (ES6)
